# Welcome To the Record Store!

### Take a REST and find all your favorite songs.
This project uses Java and the Spring Framework to power the backend of a Record Store web application.
Users can browse records, explore different categories, and manage a shopping cart while logged in.

## What This Application Does
### All Users
- View Records, Accessories and filter by categorys.
### Logged in Users
- Everything above plus add to the cart and create orders

### Side Note
The front end, database and some of the classes were provided to us to complete this project
